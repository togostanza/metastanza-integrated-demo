🔵 集合型可視化コンポーネント

集合の関係性を表現する可視化コンポーネントを準備中です。
ベン図、オイラー図などで集合間の包含関係や交差関係を表現するコンポーネントが含まれる予定です。

ベン図

複数の集合間の交集合、和集合、差集合などの関係を円形や楕円形で視覚的に表現します。

オイラー図

集合の包含関係を正確に表現するダイアグラム。ベン図よりも論理的に厳密な表現が可能です。

UpSet Plot

多数の集合の交集合を効率的に可視化するための現代的な手法です。

階層集合図

入れ子構造を持つ集合の関係性を階層的に表現するコンポーネントです。

開発状況については GitHub をご確認ください。
